# Aarogya Sathi – Community Health Companion

Aarogya Sathi is a simple web-based community health support platform that helps users access basic health information, emergency contacts, and nearby medical services. The project is designed to promote health awareness and provide quick assistance during emergencies.

## 🚀 Features

- Health tips generator
- BMI (Body Mass Index) calculator
- Emergency ambulance button
- Nearby hospital search
- Health awareness information
- Contact form for user queries
- Simple and responsive user interface

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
